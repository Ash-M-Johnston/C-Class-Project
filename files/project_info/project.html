Basic Web Server

What is a Web Server?

At it's core, a web server hosts files for users to request, usually with their browsers.

How does it work?

  1. The server opens a socket and starts listening on a specific port.
  2. A client attempts to connect by sending a GET request to the server at that port.
  3. The server processes the request and either sends back the requested file, or provides an error message if it doesn't exist.
  4. After everything is sent, the server closes the connection to the client, allowing for more connections to be opened.
